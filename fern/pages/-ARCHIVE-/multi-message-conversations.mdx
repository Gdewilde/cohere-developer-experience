There are various ways to interact with the Chat API. We covered interacting with Chat directly in the [Chat API companion doc](/docs/chat-api), here we'll discuss having a multi-message conversation with the Chat endpoint.

### Use the Previous Messages to Continue a Conversation

By specifying the `chat_history` parameter, we can provide our model with context that it can use in its replies.
