## co usage

Usage reporting

**Options**

```
  -h, --help   help for usage
```

## co usage characters

get characters usage report

```
co usage characters [flags]
```

**Options**

```
  -h, --help            help for characters
      --month string    get usage for this month, MM/YYYY
      --output string   path to output, specify "-" for stdout (default "-")
```
