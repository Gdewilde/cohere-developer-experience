## co key

Create and manage API keys.

**Options**

```
  -h, --help   help for key
```

## co key create

Create a key.

```
co key create [key_name] [flags]
```

**Options**

```
  -h, --help   help for create
```

## co key list

List your keys.

```
co key list [flags]
```

**Options**

```
  -h, --help              help for list
  -r, --include-revoked   show revoked api keys
```

## co key revoke

Revoke a key.

```
co key revoke [key_name] [flags]
```

**Options**

```
  -h, --help   help for revoke
```
